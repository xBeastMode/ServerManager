Server-Manager is used to turn on and off many things in your server.
Note :You can Customize Everything in Config.yml and Messages.yml
Here is a list of things you can manage.

| Things You Can Manage |
|-----------------------|
| Block Placing |
| Block Breaking |
| Chatting |
| Item Drop |
| Death |
| GameModeChange |
| Commands |
| PVP |
| Item Damage |
| Player Kicking |
| Sneaking |
| Achievements |
| Explosions |
| Inventory Open |
| Item Pickup |
| Item Crafting |
